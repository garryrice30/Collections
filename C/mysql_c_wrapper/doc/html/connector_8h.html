Detailed Description

MySQL C Wrapper.

Author
Garry Rice (Arprog)
Date
10/12/2019 This is a structure that simply wraps around the structures used by MySQL C Connector.

This wrapper utilizes Oracle's MySQL C Connnector. Therefore, this piece of software adhere's to their open source license.

See also
https://dev.mysql.com/doc/connector-c/en/preface.html
